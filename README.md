# Abhilasha Rehabilitation Center Premium Website

A brand-new, production-ready Next.js App Router website for Abhilasha Rehabilitation Center.

Built with:
- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Netlify-compatible deployment
- Mobile-first responsive layout

## Pages

- Home
- About
- Services
- Our Guiding Hands
- Contact
- Donate

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Netlify Deployment

1. Connect this GitHub repository in Netlify.
2. Use build command: `npm run build`
3. Publish directory: `.next`
4. Netlify will use `@netlify/plugin-nextjs` from `netlify.toml`.

## Notes

The gallery uses premium local placeholder cards and links safely to the provided Google Drive folder. It intentionally does not auto-fetch Google Drive images.
