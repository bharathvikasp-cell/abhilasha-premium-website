# Abhilasha Rehabilitation Centre Premium Website

A production-ready, premium Next.js App Router website for Abhilasha Rehabilitation Centre in Secunderabad, Telangana. The site presents the centre's therapy, education, contact, donation, team, and gallery experience with a calm NGO-focused visual system.

## Highlights

- Premium lavender and purple visual system with richer contrast
- Responsive mobile-first navigation with hamburger menu
- Therapy and education service cards with meaningful icons
- Working FormSubmit contact form, success page, and validation
- Official ARC logo optimized for web, navbar, footer, and favicon
- Google Maps CTA for the Secunderabad location
- Safe Google Drive gallery link without scraping
- Netlify-compatible Next.js deployment configuration

## Netlify Deployment

1. Connect this GitHub repository in Netlify.
2. Use build command: `npm run build`
3. Publish directory: `.next`
4. Netlify will use Node `22.14.0` and `@netlify/plugin-nextjs` from `netlify.toml`.

## Notes

The gallery uses premium local placeholder cards and links safely to the provided Google Drive folder. It intentionally does not auto-fetch Google Drive images.
